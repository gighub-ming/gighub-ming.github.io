hexo个人博客搭建步骤

注意：一定确认安装路径，在哪个盘符下的哪个文件夹，路径不要有中文特殊字符等

先看GitHub设置（最下面）

1.搭建

1.1 从“http://nodejs.cn/”官网下载nodeJS工具，并打开cmd输入node -v查看是否安装成功,并输入npm -v查看npm工具是否安装成功

2.1 使用管理员打开终端命令窗口，输入“npm install -g cnpm –registry=https://registry.npm.taobao.org”，等待安装完毕；完成后输入“cnpm”查看是否安装成功

3.1 继续输入“cnpm install -g hexo-cli”安装hexo博客插件，安装完毕后输入hexo -v 查看安装成功否

​ 输入pwd命令查看现所在的路径，输入mkdir blog 命令创建一个存放博客文件的文件夹

4.1 输入cd blog 命令进入blog文件夹，之后如果是Mac系统输入sudo hexo init，win系统输入hexo init，到此博客搭建完成了，可以输入hexo s命令开启测试端口，开启后在浏览器输入“http://localhost:4000/”可以预览博客，此网址仅本地端可预览查看，后需部署到GitHub所有人可访问

5.1 输入hexo n “我的第一篇博客”,常见了一个Markdown文档，用cd source/_posts命令进入这个文件夹，找到刚创建的文档可以用vim或typora打开进行编辑，保存完输入cd ../../回到blog文件夹，输入hexo clean清除一下，输入hexo g等待完毕

2.部署

2.1 继续输入“cnpm install –save hexo-deployer-git”安装GitHub插件

2.2如果是Mac系统直接输入hexo d进行部署，win的先输入两行命令：第一个git config –global user.email “邮箱”，第二个git config –global user.name “GitHub用户名”，之后需要输入GitHub信息，完成后再输入hexo d，这样就完成部署了。

3.GitHub的一些设置

3.1 登录www.GitHub.com登录，创建一个新仓库，名字一定要是你的"GitHub用户名.github.io“，一定要这种格式的，其他的没用，然后创建即可，以后访问就用这个名字访问
